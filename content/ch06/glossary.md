# 用語ミニ辞典（この章）

| 用語             | かんたんな説明                       |
| ---------------- | ------------------------------------ |
| 売掛金           | あとでもらう権利（資産）             |
| 買掛金           | あとで払う義務（負債）               |
| クレジット売掛金 | カード会社に対する売上代金の債権     |
| 売上返品         | 売上を取り消す科目（収益のマイナス） |
| 仕入返品         | 仕入を取り消す科目（費用のマイナス） |
