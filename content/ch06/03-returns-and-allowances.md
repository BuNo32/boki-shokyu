# 3. 返品・値引の扱い（売上／仕入）

> 売上が減るときは**売上返品**、仕入が戻るときは**仕入返品**。掛けが関わっていても、考え方は同じです。

## 売上返品（売掛金の取り消し）

- 例：掛けで売った 10,000 円分が返品に。

  | 借方科目 |   金額 | 貸方科目 |   金額 |
  | -------- | -----: | -------- | -----: |
  | 売上返品 | 10,000 | 売掛金   | 10,000 |

## 仕入返品（買掛金の取り消し）

- 例：掛けで仕入れた 6,000 円分を返品。

  | 借方科目 |  金額 | 貸方科目 |  金額 |
  | -------- | ----: | -------- | ----: |
  | 買掛金   | 6,000 | 仕入返品 | 6,000 |

> 現金で返品・値引が行われる場合も、まず<strong>掛けの取り消し</strong>が基本。そこから現金の出入りを考えます。
