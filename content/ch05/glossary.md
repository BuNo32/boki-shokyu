# 用語ミニ辞典（この章）

| 用語           | かんたんな説明                             |
| -------------- | ------------------------------------------ |
| 仕向手数料     | 送金者側が負担する振込手数料               |
| 被仕向手数料   | 受け取り側の口座から差し引かれる入金手数料 |
| 受取利息       | 銀行からもらう利息（収益）                 |
| 現金過不足     | 現金が合わないときの仮置き勘定             |
| 雑損失・雑収入 | 原因不明の不足・過剰を期末に処理する科目   |
