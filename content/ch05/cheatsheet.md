# 第5章 印刷用チートシート（A4推奨）

## 支払手段 → 使う勘定（減る側）

| 支払手段       | 使う勘定 |
| -------------- | -------- |
| 現金           | 現金     |
| 銀行振込       | 普通預金 |
| 小切手（振出） | 当座預金 |

## 資金移動（振替）

| 取引              | 仕訳                           |
| ----------------- | ------------------------------ |
| 現金→普通（預入） | （借）普通預金 /（貸）現金     |
| 普通→定期（預入） | （借）定期預金 /（貸）普通預金 |
| 定期→普通（満期） | （借）普通預金 /（貸）定期預金 |

## 利息・手数料

- 利息を受け取った → （借）普通預金 /（貸）受取利息
- 振込手数料の引落 → （借）支払手数料 /（貸）普通預金

## 4列仕訳の型

| 借方科目 | 金額 | 貸方科目 | 金額 |
| -------- | ---: | -------- | ---: |
| ＿＿＿＿ | ＿＿ | ＿＿＿＿ | ＿＿ |

> 迷ったら：**手段で右側（減る資産）を決める → 左側（何を？）を考える → 金額一致！**
