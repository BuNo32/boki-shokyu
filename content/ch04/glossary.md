# 用語ミニ辞典（この章で出てくる言葉）

| 用語     | ひらがな説明                                       |
| -------- | -------------------------------------------------- |
| 現金     | 手元のお金（レジの中）                             |
| 普通預金 | 銀行のふつうの口座（振込・引落で動く）             |
| 当座預金 | 小切手や手形で使う口座                             |
| 手段     | どうやって払った・受け取ったか（現金/振込/小切手） |
| 目的     | 何のためのお金の動きか（費用・資産など）           |
