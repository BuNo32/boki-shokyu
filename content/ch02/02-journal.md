# 2. 仕訳帳：4列の“ことば”で事実を記録

> 事実を「仕訳」ということばに翻訳します。<strong>公式は4列</strong>（借方科目・金額／貸方科目・金額）。

## 4列の型（おさらい）

| 借方科目 | 金額 | 貸方科目 | 金額 |
| -------- | ---: | -------- | ---: |
| ＿＿＿＿ | ＿＿ | ＿＿＿＿ | ＿＿ |

### 例1：現金売上 1,000円

| 借方科目 |  金額 | 貸方科目 |  金額 |
| -------- | ----: | -------- | ----: |
| 現金     | 1,000 | 売上     | 1,000 |

- 右（貸方）＝収益（売上）が増える／左（借方）＝資産（現金）が増える

### 例2：現金を普通預金に預けた 30,000円（資産の振替）

| 借方科目 |   金額 | 貸方科目 |   金額 |
| -------- | -----: | -------- | -----: |
| 普通預金 | 30,000 | 現金     | 30,000 |

- 右（出る）＝現金、左（入る）＝普通預金

### 例3：小切手を振り出して備品を買った 20,000円

| 借方科目 |   金額 | 貸方科目 |   金額 |
| -------- | -----: | -------- | -----: |
| 備品     | 20,000 | 当座預金 | 20,000 |

- **手段で右**（当座預金）、**目的で左**（備品）。この順で迷いが消えます。

## よくあるミスとコツ

- **科目の名前**に引っ張られる → まず<strong>手段＝右</strong>を先に決める
- **金額が合わない** → 最後に<strong>借方合計＝貸方合計</strong>を声に出す
- **摘要が長い** → 「何で払った？／何のため？」に要約してから書く
