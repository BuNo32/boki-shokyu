# 用語ミニ辞典（この章）

| 用語     | かんたんな説明                          |
| -------- | --------------------------------------- |
| 未収入金 | 売買以外でもらうはずの代金              |
| 未払金   | 売買以外で払うはずの代金                |
| 前払金   | 先払した前金（資産）                    |
| 前受金   | 先にもらった前受け（負債）              |
| 立替金   | 他人の代わりに払ったお金                |
| 預り金   | 他人から預かったお金（のちに渡す/納付） |
| 仮払金   | 用途未確定の前渡し                      |
| 仮受金   | 内容未確定の入金の仮置き                |
