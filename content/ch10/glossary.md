# 用語ミニ辞典（この章）

| 用語           | かんたんな説明                               |
| -------------- | -------------------------------------------- |
| 取得原価       | 買って、使える状態にするまでの総額           |
| 減価償却費     | 固定資産の価値の目減りを毎期の費用として配分 |
| 減価償却累計額 | これまで費用化した合計（資産のマイナス箱）   |
| 帳簿価額       | 取得原価 − 減価償却累計額                    |
| 定額法         | 毎期同額を費用化する方法                     |
| 修繕費         | 元に戻すための通常修理（費用）               |
| 資本的支出     | 機能向上・寿命延長など取得原価へ加算する支出 |
