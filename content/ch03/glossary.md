# 用語ミニ辞典（この章で出てくる言葉）

| 用語       | かんたんな説明                                   |
| ---------- | ------------------------------------------------ |
| 証ひょう   | 取引の証拠となる紙やデータ（レシート等）         |
| 伝票       | 仕訳を書きやすくする“指示書”（入金・出金・振替） |
| 仕訳帳     | 取引を日付順に記録する帳簿                       |
| 総勘定元帳 | 科目ごとの増減・残高を管理する帳簿               |
| 試算表     | 全科目の合計・残高をまとめた表                   |
